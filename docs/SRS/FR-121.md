# FR-121

Feature: 3.10. Mobile Application Services
Subsection: 3.10.1. Classes

## Requirement

The system SHALL allow authorized users to review attendees and update attendance outcomes for classes so that mobile users can complete operational club, event, class, attendance, and security workflows.
