# FR-126

Feature: 3.10. Mobile Application Services
Subsection: 3.10.1. Classes
Source heading: 3.10.1. Classes

## Requirement

The system SHALL allow authorized users to search, filter, page, and review classes records so that mobile users can complete operational club, event, class, attendance, and security workflows.
