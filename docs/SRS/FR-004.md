# FR-004

Feature: 3.1. User Management and Authentication
Subsection: 3.1.2. Authorization
Source heading: 3.1.2. Authorization

## Requirement

The system SHALL evaluate authorization and user-interface permissions for requested policies, actions, resources, and fields so that users, administrators, and tenants can access the platform securely and consistently.
