# FR-062

Feature: 3.6. Asset Management
Subsection: 3.6.3. Asset Checkout
Source heading: 3.6.3. Asset Checkout

## Requirement

The system SHALL validate asset checkout eligibility, rules, duplicates, or requested state changes and return clear outcomes so that asset owners and administrators can maintain controlled, auditable asset operations.
