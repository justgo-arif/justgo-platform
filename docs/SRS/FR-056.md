# FR-056

Feature: 3.5. Booking Management
Subsection: 3.5.6. Class Term
Source heading: 3.5.6. Class Term

## Requirement

The system SHALL allow authorized users to review, validate, and manage booking-related information for class term so that members, coaches, and administrators can manage participation in classes, courses, events, and sessions.
