# FR-143

Feature: 3.11. Results and Competition Management
Subsection: 3.11.1. Events
Source heading: 3.11.1. Events

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change events lifecycle state so that sports administrators and members can manage result data, rankings, competitions, and validation workflows.
