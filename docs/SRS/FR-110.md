# FR-110

Feature: 3.9. Finance and Payment Management
Subsection: 3.9.3. Installments
Source heading: 3.9.3. Installments

## Requirement

The system SHALL allow authorized finance users to review payment receipts, payment details, payment methods, refund eligibility, terminals, logs, and payment history so that organisations can manage payments, balances, recurring plans, products, and payment accounts.

## Endpoint Trace IDs

EP-285, EP-286
