# FR-150

Feature: 3.11. Results and Competition Management
Subsection: 3.11.2. Results
Source heading: 3.11.2. Results

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change results lifecycle state so that sports administrators and members can manage result data, rankings, competitions, and validation workflows.

EP-461, EP-476, EP-478
