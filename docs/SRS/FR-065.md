# FR-065

Feature: 3.6. Asset Management
Subsection: 3.6.4. Asset Credentials
Source heading: 3.6.4. Asset Credentials

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change asset credentials lifecycle state so that asset owners and administrators can maintain controlled, auditable asset operations.
