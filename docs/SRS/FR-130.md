# FR-130

Feature: 3.10. Mobile Application Services
Subsection: 3.10.3. Events
Source heading: 3.10.3. Events

## Requirement

The system SHALL allow authorized users to review attendees and update attendance outcomes for events so that mobile users can complete operational club, event, class, attendance, and security workflows.
