# FR-033

Feature: 3.2. Member Profile Management
Subsection: 3.2.5. Members
Source heading: 3.2.5. Members

## Requirement

The system SHALL allow authorized users to search, filter, page, and review members records so that members and support users can maintain accurate personal, family, contact, and preference information.
