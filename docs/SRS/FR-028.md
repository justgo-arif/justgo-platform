# FR-028

Feature: 3.2. Member Profile Management
Subsection: 3.2.4. Member Notes
Source heading: 3.2.4. Member Notes

## Requirement

The system SHALL allow authorized users to create, review, update, and remove notes for member notes so that members and support users can maintain accurate personal, family, contact, and preference information.

## Endpoint Trace IDs

EP-096, EP-097, EP-098, EP-099
