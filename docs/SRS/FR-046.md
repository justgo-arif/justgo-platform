# FR-046

Feature: 3.4. Membership Management
Subsection: 3.4.1. Memberships
Source heading: 3.4.1. Memberships

## Requirement

The system SHALL allow authorized users to download or export memberships information in a usable form so that members and organisations can understand entitlements, memberships, licences, and related downloadable records.
