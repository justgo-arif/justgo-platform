# FR-107

Feature: 3.9. Finance and Payment Management
Subsection: 3.9.2. Finance Grid View
Source heading: 3.9.2. Finance Grid View

## Requirement

The system SHALL allow authorized users to search, filter, page, and review finance grid view records so that organisations can manage payments, balances, recurring plans, products, and payment accounts.
