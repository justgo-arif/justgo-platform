# FR-108

Feature: 3.9. Finance and Payment Management
Subsection: 3.9.3. Installments
Source heading: 3.9.3. Installments

## Requirement

The system SHALL allow authorized users to download or export installments information in a usable form so that organisations can manage payments, balances, recurring plans, products, and payment accounts.
