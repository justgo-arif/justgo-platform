# FR-078

Feature: 3.6. Asset Management
Subsection: 3.6.6. Asset Licenses
Source heading: 3.6.6. Asset Licenses

## Requirement

The system SHALL provide asset licenses lookup, metadata, status, field, and configuration data required by client workflows so that asset owners and administrators can maintain controlled, auditable asset operations.
