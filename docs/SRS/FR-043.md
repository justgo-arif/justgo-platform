# FR-043

Feature: 3.3. Organisation and Club Management
Subsection: 3.3.1. Organisations
Source heading: 3.3.1. Organisations

## Requirement

The system SHALL provide organisations lookup, metadata, status, field, and configuration data required by client workflows so that members and administrators can manage club relationships, hierarchy, and transfer activities.
