# FR-076

Feature: 3.6. Asset Management
Subsection: 3.6.6. Asset Licenses
Source heading: 3.6.6. Asset Licenses

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change asset licenses lifecycle state so that asset owners and administrators can maintain controlled, auditable asset operations.

EP-199, EP-204, EP-205
