# FR-068

Feature: 3.6. Asset Management
Subsection: 3.6.5. Asset Leases
Source heading: 3.6.5. Asset Leases

## Requirement

The system SHALL allow authorized users to create, review, update, submit, approve, cancel, transfer, reinstate, or remove asset leases records so that asset owners and administrators can maintain controlled, auditable asset operations.
