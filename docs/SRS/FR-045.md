# FR-045

Feature: 3.4. Membership Management
Subsection: 3.4.1. Memberships
Source heading: 3.4.1. Memberships

## Requirement

The system SHALL allow authorized users to create, submit, or process memberships records while preserving required business data so that members and organisations can understand entitlements, memberships, licences, and related downloadable records.
