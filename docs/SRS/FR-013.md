# FR-013

Feature: 3.1. User Management and Authentication
Subsection: 3.1.9. Tenants
Source heading: 3.1.9. Tenants

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change tenants lifecycle state so that users, administrators, and tenants can access the platform securely and consistently.
