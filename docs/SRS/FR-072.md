# FR-072

Feature: 3.6. Asset Management
Subsection: 3.6.5. Asset Leases
Source heading: 3.6.5. Asset Leases

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change asset leases lifecycle state so that asset owners and administrators can maintain controlled, auditable asset operations.
