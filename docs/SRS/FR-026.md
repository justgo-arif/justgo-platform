# FR-026

Feature: 3.2. Member Profile Management
Subsection: 3.2.3. Member Family
Source heading: 3.2.3. Member Family

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change member family lifecycle state so that members and support users can maintain accurate personal, family, contact, and preference information.

## Endpoint Trace IDs

EP-088, EP-093, EP-095
