# FR-091

Feature: 3.6. Asset Management
Subsection: 3.6.9. Asset Registers
Source heading: 3.6.9. Asset Registers

## Requirement

The system SHALL allow authorized users to search, filter, page, and review asset registers records so that asset owners and administrators can maintain controlled, auditable asset operations.
