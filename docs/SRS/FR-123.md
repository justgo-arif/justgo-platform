# FR-123

Feature: 3.10. Mobile Application Services
Subsection: 3.10.1. Classes
Source heading: 3.10.1. Classes

## Requirement

The system SHALL allow mobile users to review class lists, sessions, occurrences, tickets, member details, booking details, eligibility, and payment-related class context so that mobile users can complete operational club, event, class, attendance, and security workflows.
