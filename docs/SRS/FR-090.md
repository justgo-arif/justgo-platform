# FR-090

Feature: 3.6. Asset Management
Subsection: 3.6.9. Asset Registers
Source heading: 3.6.9. Asset Registers

## Requirement

The system SHALL provide asset registers lookup, metadata, status, field, and configuration data required by client workflows so that asset owners and administrators can maintain controlled, auditable asset operations.

EP-229, EP-240
