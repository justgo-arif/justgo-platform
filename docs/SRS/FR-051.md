# FR-051

Feature: 3.5. Booking Management
Subsection: 3.5.1. Booking Catalog
Source heading: 3.5.1. Booking Catalog

## Requirement

The system SHALL allow authorized users to review, validate, and manage booking-related information for booking catalog so that members, coaches, and administrators can manage participation in classes, courses, events, and sessions.
