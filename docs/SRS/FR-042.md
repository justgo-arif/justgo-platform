# FR-042

Feature: 3.3. Organisation and Club Management
Subsection: 3.3.1. Organisations
Source heading: 3.3.1. Organisations

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change organisations lifecycle state so that members and administrators can manage club relationships, hierarchy, and transfer activities.

## Endpoint Trace IDs

EP-121, EP-129
