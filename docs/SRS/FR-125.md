# FR-125

Feature: 3.10. Mobile Application Services
Subsection: 3.10.1. Classes
Source heading: 3.10.1. Classes

## Requirement

The system SHALL provide classes lookup, metadata, status, field, and configuration data required by client workflows so that mobile users can complete operational club, event, class, attendance, and security workflows.
