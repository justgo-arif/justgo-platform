# FR-017

Feature: 3.1. User Management and Authentication
Subsection: 3.1.11. Users
Source heading: 3.1.11. Users

## Requirement

The system SHALL allow authorized users to update, remove, approve, cancel, archive, or otherwise change users lifecycle state so that users, administrators, and tenants can access the platform securely and consistently.
